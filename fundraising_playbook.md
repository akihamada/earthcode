# EARTH CODE 資金調達プレイブック（v1.0）

> "Raise at the speed of thought, not at the speed of decks."
> ピッチデックを磨くより、**物理的に動くMVPを作って殴る**方が速い。

本プレイブックは、EARTH CODE のシード調達を **最短90日** でクローズするための行動指針である。

---

## 1. 調達戦略の全体像

### 1.1 3トラック並列モデル

```
  Day 0                Day 30              Day 60             Day 90
  │                    │                   │                  │
  ├─ Track A ──────────┤                                      │
  │  Angel Bridge      │ クローズ ¥20–30M                      │
  │  (J-KISS)          │ → サウナ小屋 着工                     │
  │                    │                                      │
  │                    ├─ Track B ───────────────────────────┤
  │                    │  Seed Round (J-KISS)                │
  │                    │  キックオフ → パイプライン構築 → LOI → クローズ ¥50–80M
  │                    │                                      │
  ├─ Track C ─────────────────────────────────────────────────┤
  │  Grants (Non-dilutive)                                   │
  │  NEDO / 環境省 / 経産省 → 申請 → 採択 ¥10–30M               │
```

### 1.2 なぜ3トラックか
- **Track A (エンジェル)**: デモサイト着工という **unblocker**。意思決定が速い個人投資家から短期間で集める。
- **Track B (シード本体)**: 機関投資家（VC）は意思決定に時間がかかるため、**Aで物理を作ってから**パイプラインに流す。
- **Track C (非希釈)**: 希釈ゼロで研究実証費用を賄う。バリュエーション向上に直結。

3つを**同時に走らせる**ことで、どれか1つが遅延しても全体は止まらない。

## 2. Track A — エンジェル・ブリッジ（¥20–30M / 30日）

### 2.1 ストラクチャ
- **J-KISS 2.0**（ディスカウント 20% / バリュエーションキャップ ¥800M）
- 1口 ¥5M、4–6名想定
- リーガル: Coral Capital の J-KISS テンプレートをそのまま採用（弁護士費用を最小化）

### 2.2 ターゲット投資家プロファイル
| セグメント | 特徴 | 説得材料 |
|---|---|---|
| **建築・不動産系経営者** | 土に対する直感的理解がある | マテリアル・サンプルと特許 |
| **サステナビリティ系エンジェル** | ESG文脈で高いWTP | CO2削減数値とカーボンクレジット |
| **連続起業家（ディープテック経験者）** | MVP重視 | 90日で建つサウナ |
| **軽井沢の別荘オーナー** | 現地に共感、将来の顧客化も | 現地ツアー招待 |
| **JR東海 / 大手ゼネコンOB** | BFRPの技術系譜を理解 | research_proposal.md § 6 |

### 2.3 アプローチ
1. **Week 0**: 上記カテゴリから各 2–3名 をリストアップ（計 10–15 名）
2. **Week 1**: ウォームイントロ経由で 1:1 ミーティング打診
3. **Week 2**: ピッチ + 浅沼組サンプル持参、J-KISS 条件提示
4. **Week 3**: 2回目面談 & SAFE 送付
5. **Week 4**: クロージング

### 2.4 決定的な殺し文句
> 「このラウンドの¥20Mは、**軽井沢で実際にサウナ小屋を建てる物理費用**です。60日後にはあなたを招待して入っていただきます。その時点でのシード本体ラウンドは、このラウンドの2倍以上のバリュエーションになります。」

**意図**: 使途の明確さ + 短期的アップサイドの明示 + FOMO

## 4. Track C — 非希釈資金（補助金・助成金）

### 4.1 優先ターゲット
| 制度 | 金額感 | 申請時期 | 申請主体 |
|---|---|---|---|
| NEDO「カーボンニュートラル実現に向けた技術開発」 | ¥10–30M | 通年 | EARTH CODE 単独 or 浅沼組との共同 |
| 環境省「脱炭素先行地域」連携 | 地域事業依存 | 年2回 | 軽井沢町との連携が鍵 |
| 経産省「ものづくり補助金」 | ¥10–30M | 年数回 | 設備投資（3Dプリンタ等） |
| JST CREST / さきがけ | 研究費 | 年1回 | 藝大金田研との連携 |
| 林野庁「木材利用促進」 | ¥5–10M | 年1回 | 竹材使用が該当 |
| 長野県 中小企業支援金 | ¥2–5M | 通年 | 軽井沢所在地要件 |

### 4.2 申請代行
- 専門家（認定支援機関の行政書士・中小企業診断士）に申請支援を外注
- 成功報酬15%前後が相場
- 自社リソースを投入しない運用が鉄則

### 4.3 連携パートナー戦略
補助金は単独申請より共同申請の方が採択率が上がる:
- 浅沼組技術研究所（大手ゼネコン）
- 藝大金田研（アカデミア）
- Arup（構造解析）
- 軽井沢町（地方自治体連携）

## 6. ピッチ配信戦略

### 6.1 ピッチ・チャネル優先順位
1. **1対1面談**（最強）: コンバージョン率が最も高い
2. **Demo Day in 軽井沢**（月1回）: 複数社同時、競合環境
3. **スタジオ訪問ピッチ**: シードVC主催のピッチイベント
4. **オンラインピッチ**（最弱）: 時間節約のためのスクリーニング用途のみ

### 6.2 コミュニケーション速度
- 投資家からの連絡には **24時間以内** に返信
- ミーティング後 **48時間以内** にフォローアップ資料送付
- タームシート交渉は **7日以内** に意思決定を要求

### 6.3 競合環境の作り方（Non-sleazy FOMO）
- 「現在2社とタームシートを交渉中、来週中に意思決定予定」（**事実ベース**で伝える）
- 「Demo Day に他社も招待しています」（透明性を保つ）
- 「エンジェルラウンドはすでにオーバーサブスクライブ」（実績を示す）

## 8. 創業者のフルタイム活動配分（90日）

| 活動 | 配分 | 備考 |
|---|---|---|
| 投資家面談・ピッチ | 40% | 最優先 |
| デモサイト設計・監修 | 20% | Web 1–4 集中 |
| データルーム整備・DD対応 | 15% | 週末に集中 |
| パートナー調整（浅沼組・金田研） | 10% | 定例会議 |
| 採用（COO候補） | 10% | 並行 |
| 研究・論文 | 5% | 委任可 |

**原則**: 90日間は創業者の時間の半分以上を「投資家との直接接触」に使う。デザイン・設計・施工監理は**委任できる相手を先に確保**する。

## 9. リスクと対応

| リスク | 対応 |
|---|---|
| エンジェル・ブリッジが集まらない | 創業者の個人資金 or 土地担保ローンでサウナ着工 → 後日リカバー |
| VCのDDが長期化 | 「DDの間に建設が進む」ことをむしろ武器化 |
| 軽井沢の許認可が遅れる | 仮設・可搬式・東京での先行ミニチュア建設にピボット |
| リードVCが降りる | Tier 2 への即切替リスト準備 |
| バリュエーション低下圧力 | 非希釈（補助金）比率を増やして対応 |
